## ðŸ  **Window (CTk)**

**What:** Main app window.
**Rules:**

* Only one per app.
* Must call `mainloop()` at end.
* Use `set_appearance_mode()` and `set_default_color_theme()` before creating it.

**Main attributes:**

* `title("...")` â€“ sets window name.
* `geometry("800x600")` â€“ size.
* `resizable(x, y)` â€“ allow resizing.
* `iconbitmap("icon.ico")` â€“ set icon (Windows only).

**Example:**

```python
import customtkinter as ctk
ctk.set_appearance_mode("dark")
ctk.set_default_color_theme("blue")
root = ctk.CTk()
root.title("My App")
root.geometry("800x600")
root.mainloop()
```

## ðŸ“¦ **Frame (CTkFrame)**

**What:** Container to group widgets.
**Rules:**

* Can mix `pack`, `grid`, `place` across different frames, **not inside the same one**.
* Use for sections like forms, headers, or lists.

**Main attributes:**

* `corner_radius` â€“ round corners.
* `fg_color` â€“ background color (None = transparent).
* `border_width` â€“ outline thickness.
* `border_color` â€“ outline color.

**Example:**

```python

frame.grid(row=0, column=0, padx=10, pady=10)
```

## âœï¸ **Entry (CTkEntry)**

**What:** Single-line input.
**Rules:**

* Attach a `StringVar` for reading/writing values easily.

**Main attributes:**

* `textvariable=var` â€“ binds variable.
* `placeholder_text="Enter amount"` â€“ faded hint.
* `width` â€“ entry size.

```python

```

## ï¿½ï¸ **Dialog Box (CTkToplevel for Input)**

**What:** Modal popup for user input (e.g., add expense details).
**Rules:**

* Use `CTkToplevel` for custom input dialogs.
* Make modal with `grab_set()` and `wait_window()`.

**Example code:**

```python
class InputDialog(ctk.CTkToplevel):
    def __init__(self, parent):
        super().__init__(parent)
        self.title("Add Expense")
        self.geometry("300x200")
        self.grab_set()
        
        self.label = ctk.CTkLabel(self, text="Amount:")
        self.label.pack(pady=10)
        self.entry = ctk.CTkEntry(self)
        self.entry.pack(pady=10)
        
        self.ok_btn = ctk.CTkButton(self, text="OK", command=self.on_ok)
        self.ok_btn.pack(side="left", padx=20)
        self.cancel_btn = ctk.CTkButton(self, text="Cancel", command=self.on_cancel)
        self.cancel_btn.pack(side="right", padx=20)
        
        self.result = None
    
    def on_ok(self):
        self.result = self.entry.get()
        self.destroy()
    
    def on_cancel(self):
        self.result = None
        self.destroy()

# Usage in main app:
def open_dialog():
    dialog = InputDialog(app)
    app.wait_window(dialog)
    if dialog.result:
        print(f"Input: {dialog.result}")
```

**Main attributes:**

* `grab_set()` â€“ blocks parent window.
* `wait_window(dialog)` â€“ waits for dialog close.
* Access result via `dialog.result`.

## âš™ï¸ **Layout Managers**

### `.pack()`

* Simple vertical or horizontal stacking.
* Options: `side`, `fill`, `expand`, `padx`, `pady`.

**Example:**

```python
button1 = ctk.CTkButton(root, text="Button 1")
button1.pack(side="top", fill="x", padx=10, pady=5)
button2 = ctk.CTkButton(root, text="Button 2")
button2.pack(side="top", fill="x", padx=10, pady=5)
```

### `.grid()`

* Place widgets in a table.
* Options: `row`, `column`, `sticky`, `padx`, `pady`.
* Configure columns: `frame.grid_columnconfigure(col, weight=1)` for stretch.

**Example:**

```python
label1 = ctk.CTkLabel(root, text="Label 1")
label1.grid(row=0, column=0, sticky="w", padx=5, pady=5)
label2 = ctk.CTkLabel(root, text="Label 2")
label2.grid(row=0, column=1, sticky="e", padx=5, pady=5)
```

### `.place()`

* Exact positioning (`x`, `y`). Not usually needed.

**Example:**

```python
button = ctk.CTkButton(root, text="Button")
button.place(x=50, y=50)
```

**âš  Rule:**

> Never mix `pack()` and `grid()` in the same parent widget.
